# Database Registry Template

Track your Notion database IDs and schemas here.

## Example Database

- **Database ID**: `your-database-id-here`
- **Purpose**: Description of what this database tracks
- **Key writable fields**:
  - Name (title)
  - Status (status)
  - Tags (multi_select)
- **Read-only fields**: Created time, Last edited time

## How to Find Database IDs

1. Open the database in Notion
2. Click "..." → "Copy link"
3. The ID is the 32-character string in the URL (without hyphens)
4. Example: `https://notion.so/workspace/22ce1b43239381a49443e32e71142e0d` → ID is `22ce1b43239381a49443e32e71142e0d`
